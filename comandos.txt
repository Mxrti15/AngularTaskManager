# AngularTaskManager
Comprovar version node
node --verssion
En caso de estar desactualizado
https://nodejs.org/en/ 

instalacion Angular: -g(paquete global)
npm install -g @angular/cli
Comprovar version Angular
ng version

Iniciamos el proyecto con su nombre
ng new angular-task-manager
Aquí te preguntarà un par de cosas..
Entramos al proyecto
cd angular-task-manager
Ejecutamos el servidor
ng serve

Generar componentes
ng generate component pages/home

Generar servicios
ng generate service task
